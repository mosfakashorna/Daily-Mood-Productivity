# Daily Mood & Productivity
This is a simple Python command-line program that allows users to track their daily mood and productivity.

This program records daily entries and calculates the average mood and productivity over time.

# Features
- Log daily mood (1-10)
- Log productivity score (1-10)
- View mood and productivity history.
- Calculate average mood score
- Calculate productivity score
- Simple command-line menu system

# How It Works
This program stores mood and productivity data using Python lists. 
User can add entries each day and view history or calculate averages.

# Skills Used
- Python
- Lists
- Loops
- Conditional statements
- Basic data tracking
- Arithmetic
